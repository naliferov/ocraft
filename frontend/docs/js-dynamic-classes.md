**Шемсединов — динамические классы в JS** ([видео](https://www.youtube.com/watch?v=o9gYlSou0vY)) - классы с именем в рантайме для метапрограммирования / DSL / выразительного доменного кода.

**трюк - класс с вычисляемым именем:**

- классы - объекты первого класса → создавай/возвращай их в рантайме
- анонимный класс → `name === ''`; присвоенный переменной → берёт имя переменной
- чтобы задать имя из строки - оберни анонимный класс в объект с вычисляемым ключом, потом вытащи:
  ```js
  const { [name]: cls } = { [name]: class {} } // cls.name и cls.prototype.constructor.name === name
  ```
- 3 равнозначные формы - контейнер / шортхенд / деструктуризация (он любит контейнер)
- `Object.defineProperty` тоже работает, но не стоит - засоряет `Object.keys` + `console.log`; имя класса должно оставаться non-writable + non-enumerable
- **TypeScript так не умеет** (статическая типизация vs классы, созданные в рантайме)

**зачем:** чистые стек-трейсы + `console.log`, и классы, прочитанные из схемы в рантайме, ведут себя как написанные вручную.

**юзкейс - `struct` (его библиотека `metautil`):**

- `struct.immutable(defaults)` / `struct.mutable(defaults)` → строит класс с динамическим именем
- **defaults заодно задают рантайм-типы** (`typeof` каждого дефолта; хелпер `typeOf` добавляет null + array) → присваивание не того типа кидает ошибку
- **immutable:** `Money.create({ amount, currency })` - поля не изменить
- **mutable:** `task.create(…)`, дальше `task.update(…)` / `task.status = 'running'` меняют на месте; `toObject()` → обычный объект
- **`fork`** = отсоединённая копия · **`branch`** = переопределяет часть полей, остальное наследует через **цепочку прототипов** (общие поля хранятся один раз) - оба работают с mutable + immutable (напр. тема → base / compact / accessible, сцеплены прототипами)
- суть: защитить доменные объекты от мутаций, когда передаёшь их по ссылке между модулями
- ещё добавил односвязный список только-на-добавление (остаётся immutable для держателей старых ссылок)

**претензия:** stdlib JS бедна на структуры данных (только Array/Object/Set/Map/Weak\*/TypedArrays) - хочет очередь, стек, связные/циклические списки и т.д.; предложение records & tuples так и не приняли, поэтому он сделал своё.

**вывод:** классы с именем в рантайме - чистый инструмент для DSL + выразительного доменного кода.
