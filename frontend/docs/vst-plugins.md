# vst plugins

## Тир S — мастхев

- **Serum** — синтезатор №1 в электронной музыке.
- **FabFilter Pro-Q 3** — эталонный EQ, стандарт индустрии.
- **Soothe 2** — динамически убирает конфликтные / резкие частоты (резонанс-саппрессор).
- **Diva** (u-he) — лучшая эмуляция аналогового синтезатора, звучит как железо.

## Тир A — отлично

- **Thermal** (Output) — сатуратор, добавляет теплоту и характер.
- **OTT** (Xfer) — мультибанд-компрессор (есть стоковый и сторонний).
- **Valhalla Supermassive** — бесплатный реверб с огромным набором пресетов.
- **FabFilter Saturn 2** — сатурация и дисторшн с хирургической точностью.

## Тир B — опционально

- **Cableguys ShaperBox** — модуляция всего: ритмичный гейтинг, тремоло и т.д.
- **Kickstart** (Nicky Romero) — простой сайдчейн в два клика.

## Тир C — очень ситуативно

- **Endless Smile** — макросы эффектов для билдапов.
