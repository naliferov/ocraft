generative art — livecoding stack

Text-driven generative audio + visuals. All of it is code, so it iterates fast and pairs well with an AI collaborator.

Strudel (audio)

TidalCycles ported to JavaScript / the browser (strudel.cc). A pattern language whose sample functions are literally Prefuse-style chopping primitives:

  • chop, striate, slice, bite, chew — granular/chop a loaded sample
  • degradeBy, sometimes, somecycles, rand, irand, choose, wchoose — first-class randomization
  • mini-notation for Euclidean / polyrhythm / ratchets

Load a sample and one line gives randomized, beat-locked granular chopping:

s("durov").slice(16, irand(16).segment(8)).sometimesBy(.3, rev).chop(4)

Runs on its own web-audio engine (superdough), or sends to SuperDirt / SuperCollider for sample playback. Likely the shortest path to the chopped-vocal aesthetic, and it is pure text.

SuperCollider (audio)

The heavyweight underneath: a full synthesis language + server (scsynth). SynthDefs, Pbind patterns, deep granular (TGrains, GrainBuf, Warp1), buffer DSP, total control. Notably it is the engine room beneath TidalCycles/Strudel: their default sample backend, SuperDirt, runs on SuperCollider. Steeper curve, highest ceiling.

Hydra (video)

Browser-based livecoding visuals (Olivia Jack). Analog-synth-style functional chains on WebGL:

osc(150, -0.1).rotate(1.57).out(o0)
osc(20, 0.1, 0.8).kaleid(5).modulate(noise(3)).out()

Audio-reactive via FFT (a.fft[0]), and can ingest external video / webcam / screen (s0.initVideo(...)) — so footage can be fed in and glitched generatively.

How they connect

  • TidalCycles/Strudel → SuperDirt (SuperCollider) for samples.
  • Strudel ↔ Hydra integrate; Flok is a collaborative editor hosting Strudel + Hydra + SC together.
  • OSC is the universal glue — links SC, Tidal, Max, and Ableton (via AbletonOSC). See [[prefuse-style-glitch-chopping-tools]].
