хармс, вєдєнський, костянтин вагінов
андрій платонов — «вʼязкий, остранений» язик; другий полюс порівняння
лоренс стерн, «трістрам шенді» — європ. дід манери: дигресія як структура, типографічна гра, автор влазить у власний текст

російський андеграунд / маргінес
саша соколов («школа для дураков», «палисандрия», «между собакой и волком») — найближчий за фактурою: щільна каламбурна орнаментальна «проза поета», логорея, самоміфологізація; точний матч тканини → починай звідси
алексей ремізов — орнаментальний сказ, архаїчна орфографія, рукотворні книги-обʼєкти з власними малюнками; майже прямий предок ханона по «книга-обʼєкт»
дмитрий галковский, «бесконечный тупик» — гіпертекст приміток-до-приміток, самвидав культовий обʼєкт, автор-персонаж, параноїдальна ерудиція; найближче по «компендіум»
венедикт єрофєєв, «москва–петушки» — іронічна пʼяна ерудована дигресія, самвидав

український андеграунд
майк йогансен — полімат (поет/прозаїк/перекладач/критик/лінгвіст); «подорож ученого доктора леонардо…» — метапроза, автор грає з власним текстом; історичний укр. матч
юрій іздрик, «воццек» — «змагання з мовою», текст-лабіринт сторінок/рядків/слів/літер/звуків, множинність «я»; станіславівський феномен, часопис «четвер»; щільний, каламбурний, самоміфологічний; заходити з «АМтм», не з «воццека»
емма андієвська — емігрантський сюрреалізм, неологістична герметична щільна проза

британський / ірландський
алаздер ґрей («lanark», «poor things») — типографічна гра, самоілюстровані книги-обʼєкти, автор-інтрузія («index of plagiarisms»), самоміфологізація; сильний по «книга-обʼєкт», читабельний
фленн оʼбрайен («at swim-two-birds», «the third policeman») — метафікція, виноски-як-окремий-роман (de selby), комічна дигресія, ерудований абсурд; найлегший вхід, смішно
ентоні берджесс — логофіл, винайдене арго (nadsat у «заводному апельсині»), щільна звукова проза

американський
вільям ґасс («the tunnel», «willie masters' lonesome wife») — проза-як-обʼєкт, типографіка, одержимість звуком слова, лексична щільність; «щільність» + «книга-обʼєкт»
александр теру («darconville's cat») — лексичний максималізм, списки, надмір ерудиції; найчистіший матч по щільності словника
девід марксон (пізні «notecard novels»: «wittgenstein's mistress», «reader's block») — фрагмент-компендіум ерудиції, самореферентність; «компендіум»
марк z. данілевський, «house of leaves» — типографічний лабіринт, виноски-в-виносках, книга-як-будинок; сучасний матч по «книга-обʼєкт»
